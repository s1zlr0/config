Эмулятор командной строки VFS
1. Общее описание
Эмулятор командной строки VFS (Virtual File System) - это графическое приложение, имитирующее работу командной оболочки UNIX-подобных операционных систем. На первом этапе реализован минимальный прототип с базовым функционалом REPL (Read-Eval-Print Loop).
